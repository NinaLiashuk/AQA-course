В папке main/..hw1 три файла : интерфейс ICalculator, класс Calculator, реализующий интерфейс и класс Main.
 
В папке test/..hw1 файл CalculatorTest с юнит-тестами.


Отличия абстрактного класса от интерфейса:

1. Абстрактный класс содержит поля и методы (то есть описывает состояние и поведение), интерфейс - только методы.

2. Методы интерфейса не содержат реализацию, абстрактный класс может иметь методы с реализацией (абстрактные методы не имеют реализации).

3. Абстрактный класс может иметь конструкторы, интерфейс - нет.

4. Какой либо подкласс может наследоваться только от одного абстрактного класса. При этом интерфейс позволяет множественное наследование, то есть один подкласс может имплементировать несколько интерфейсов.

5. Абстрактный класс может наследовать другой класс и реализовывать интерфейсы, а интерфейс может только наследовать другие интерфейсы.

6. Мы можем запустить абстрактный класс, если он имеет метод main(). Мы не можем запустить интерфейс, потому что у него не может быть реализации метода main().

7. Класс-наследник от интерфейса обязан реализовать все его методы, класс -наследник от абстрактного класса обязан реализовать только абстрактные методы.
