В папке dto содержатся классы User, Account, Transaction.

В папке querry содержатся классы с SQL - запросами в базу данных для получения или записи данных.

В папке service содержатся классы, в которых создаются объекты классов User, Account, Transaction.

В папке util содержится класс ConsoleMenu, содержащий все методы из других папок, для взаимодействия пользователя с базой. И класс Main.
